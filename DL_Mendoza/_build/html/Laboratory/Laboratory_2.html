import numpy as np

#  Given Inputs
x = np.array([[1], [0], [1]])   # column vector
y = np.array([[1]])

# Hidden and output weights
W_hidden = np.array([
    [0.2, -0.3],
    [0.4,  0.1],
    [-0.5,  0.2]
])

W_output = np.array([[-0.3, -0.2]])

# Bias (Œ∏)
theta = np.array([[-0.4], [0.2], [0.1]])

# ReLU activation function
def relu(z):
    return np.maximum(0, z)

# Hidden layer computation
Z_hidden = np.dot(W_hidden.T, x) + theta[:2]   # Œ∏‚ÇÅ and Œ∏‚ÇÇ for hidden layer
print("Hidden layer weighted sums (Z_hidden):\n", Z_hidden)

# Apply ReLU activation
A_hidden = relu(Z_hidden)
print("\nHidden layer activations (A_hidden):\n", A_hidden)

# Output layer weighted sum (with Œ∏‚ÇÉ bias)
Z_output = np.dot(W_output, A_hidden) + theta[2]  # Œ∏‚ÇÉ for output bias
y_pred = Z_output
print("\nPredicted Output (yÃÇ):\n", y_pred)

In the result, we can see that after multiplying the inputs with their weights and adding the biases, we got values of -0.7 and 0.1 for the hidden layer. When we used the ReLU activation function, the -0.7 became 0 because ReLU removes negative numbers, and the 0.1 stayed the same. So, the hidden layer output became [0, 0.1]. This means only one neuron in the hidden layer was active. These values were then passed to the output layer, which calculated the final predicted value of 0.08. The correct or target output should be 1, but our model only predicted 0.08, which means it‚Äôs still far from accurate. This happens because the weights and biases are not yet trained, in other words the model still needs learning or adjustments to get closer to the correct answer.

# Compute error
E = 0.5 * (y - y_pred)**2
print("\nError (E):\n", E)

Error (E):

Here, I calculated how far the predicted output was from the actual target using the formula for error. After plugging in the values, the error came out to 0.4232. This number shows that there‚Äôs still a noticeable difference between what the network predicted and what the real answer should be. A smaller error means better performance, so this result tells us the model still needs improvement to make more accurate predictions.
